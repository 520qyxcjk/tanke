# 🎮 坦克大战 — Battle City

一款基于 HTML5 Canvas 的经典坦克大战游戏，纯浏览器端运行，无需任何依赖。

## 在线游玩

直接打开 `index.html` 即可开始游戏。

## 操作说明

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 移动坦克 |
| `空格` / `J` | 射击 |
| `P` | 暂停/继续 |
| `Enter` | 开始游戏 / 重新开始 |

## 游戏特性

- **5个关卡**：从经典布局到迷宫，难度递增
- **敌方AI**：敌人会主动寻找玩家基地并射击
- **波次系统**：每关有多波敌人，全部消灭后进入下一关
- **基地保护**：保护底部的鹰旗不被摧毁
- **多样地形**：可破坏的砖墙和不可破坏的钢墙
- **爆炸特效**：粒子爆炸动画效果

## 项目结构

```
├── index.html    # 完整游戏（单文件，含所有HTML/CSS/JS）
├── README.md     # 本文件
```

## 技术实现

- 纯原生 JavaScript + Canvas 2D API
- 像素级碰撞检测
- 基于瓦片的地图系统
- 游戏循环使用 `requestAnimationFrame`
- 响应式布局
